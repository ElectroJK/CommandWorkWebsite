# ProjectHub

A modern **React.js web application** featuring theme switching, responsive UI, modular components, and a clean project structure.

---

##Features
- **Dark/Light theme** stored in `localStorage`
- **Responsive layout**
- **Reusable UI components**
- **Organized folder structure**
- **Lightweight and fast**

---

##Tech Stack
- React.js  
- React Router  
- Context API  
- JavaScript (ES6+)  
- CSS / Tailwind (if used)
